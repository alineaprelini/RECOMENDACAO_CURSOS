# Recomendação de cursos na Coursera
Este trabalho, desenvolvido no âmbito da disciplina Projeto Aplicado III, teve como objetivo construir um sistema de recomendação de cursos online utilizando técnicas de Machine Learning, com ênfase na personalização de conteúdos educacionais. A crescente oferta de cursos em plataformas digitais gera sobrecarga informacional, tornando a filtragem inteligente um recurso essencial para orientar os usuários.
Para isso, adotou-se uma abordagem de filtragem baseada em conteúdo, na qual as características dos cursos foram representadas por vetores numéricos obtidos pela aplicação da técnica TF-IDF (Term Frequency – Inverse Document Frequency) sobre a coluna skills da base de dados. O modelo de similaridade foi construído com o algoritmo K-Nearest Neighbors (KNN) e a métrica do cosseno, permitindo identificar e recomendar cursos semanticamente relacionados.
O projeto seguiu um pipeline estruturado composto por: análise exploratória dos dados, tratamento e pré-processamento, vetorização textual, treinamento do modelo, geração de recomendações e avaliação de desempenho. As métricas Precision@K e Recall@K foram empregadas para mensurar a qualidade das recomendações, complementadas por análise qualitativa.
Os resultados demonstraram que o modelo é capaz de sugerir cursos coerentes e semanticamente relacionados ao item de referência, evidenciando o potencial da abordagem adotada para aplicações em ambientes educacionais digitais. Limitações identificadas, como ausência de personalização por usuário e dependência exclusiva da coluna skills, apontam direções para trabalhos futuros, incluindo a implementação de modelos híbridos e técnicas de aprendizado profundo.
